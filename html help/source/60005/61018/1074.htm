Avoid using signals management Functions

Rationale

Failing to properly handle various signals would likely cause your application to terminate, when it receives such signals. In addition, the use of signal management functions slows down the application.

Description

This rule searches for C/C++ functions that call signal management functions like "signal" or "raise".

Remediation

Avoid signal management when possible. Secure signal management with proper signal masking (use of "sigaction()")
