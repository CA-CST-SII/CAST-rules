Rationale

Unreferenced views may be the symptom of Dead Code. And dead Code must be avoided as it makes source code less readable and increases the cost of the software maintenance. A view is flagged as unreferenced when it is not called. Warning: Unreference code can also be the symptoms of missing code (code not included in the source code analysis) and / or can be the symptom of use of polymorphism.

Description

List of schema views that are not called

Remediation

inspect code, check it is truly unnecessary. Remove views that are not used.

Output

this report lists all unreferenced views . It provides the following information: - View name - View fullname

Total

total is number of views stored in the schema
