Use WHEN OTHERS in exception management

Rationale

If not handled, exceptions can lead to unwanted program behaviour.

Description

WHEN OTHERS must be present in exception management.

Remediation

Add a WHEN OTHERS clause in the exception trapping mechanism.

Sample
EXCEPTION
     WHEN DUP_VAL_ON_INDEX THEN
        raise_application_error (-20001,'You have tried to insert a duplicate value.');

END;
Remediation sample
EXCEPTION
     WHEN DUP_VAL_ON_INDEX THEN
        raise_application_error (-20001,'You have tried to insert a duplicate value.');

    WHEN OTHERS THEN
        raise_application_error(-20001,'An error was encountered - '||SQLCODE||' -ERROR- '||SQLERRM);

END;
