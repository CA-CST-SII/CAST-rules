Weight in Parent Tech. Criterion

Rationale

Unreferenced methods like any other example of dead code makes maintenance more difficult and more expensive (modification of unused code).

Description

This metric retrieves all ABAP methods that are not referenced i.e. not called.

Remediation

Check if the method is going to be used. If not, then remove it from the class.

Output

This report lists all ABAP methods that are not referenced. It provides the following information: ABAP Method full name

Total

Number of ABAP Methods
