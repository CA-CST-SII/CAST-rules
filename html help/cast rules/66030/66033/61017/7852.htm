Rationale

For readability aspects, naming conventions must be respected.

Description

Names of Views should start with X or by 'ZV' for SAP database views. This reports shows all the database views not starting with the prefix. The prefix value is a parameter and can be changed at will.

Remediation

Change name according to the naming convention.

Reference

Enhancing the Quality of ABAP Development, Meijs-Krouwels-Heulmans-Sommen, SAP Press ISBN 1-59229-030-2

Sample
The following name does not respect naming conventions for SAP database views: ZMATERIAL
Remediation sample
The name should be ZVMATERIAL or ZV_MATERIAL.
