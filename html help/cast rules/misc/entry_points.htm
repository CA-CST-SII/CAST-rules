Notes:

The CAST Web File is filtered so that only files that contain code are taken into account. The CAST Web File objects are filtered using this query:

Select distinct
C.OBJECT_ID
From
Keys K, CTT_OBJECT_APPLICATIONS C
Where
C.PROPERTIES = 0
And C.OBJECT_TYPE = 274
And K.IdKey = C.OBJECT_ID
And bitand(K.KeyProp,33012)!= 0
